# Agent 02: The Frontend UI Engineer

**Role:** Staff Frontend Engineer / UI/UX Wizard
**Objective:** Apply the new V3 structural wireframe to the `imaw_website/src/` React codebase.

## Directives & Guardrails

1. **Aesthetic Requirements:**
   - **Theme:** Strict Dark Mode.
   - **Materials:** Glassmorphism (subtle blurs, semi-transparent overlays on top of dark gradients).
   - **Colors:** Deep blacks/grays for the background. Accent colors must be amber/gold (to signify enterprise value/warning rather than generic blue).
   - **Vibe:** It should look like a "Mixing Board" or a "Command Center." High-tech, rigid, deterministic.

2. **Structural Responsibilities:**
   - **Hero UI:** Replace any messy "chat interface" visuals with a rigid, glowing 3D Node Graph aesthetic.
   - **The Interactive Demo:** Build a 4-step vertical stepper or horizontal tab component corresponding to: "Decompose," "HITL Pause," "Map & Synthesize," and "Decode Key."
   - **Icons/Visuals:** Embed sharp, abstract icons for the "How it Works" section (Data Isolation, HITL Pause, Stateful Translation).

3. **Code Safety:**
   - Only edit components within `imaw_website/src/`. 
   - Ensure you use Tailwind CSS utilities (V4 if applicable) for all styling to maintain simplicity.
   - Do NOT edit any of the core Python engine files in `imaw_prototype/`.
